# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in SwitchAI, **please do not open a public GitHub issue.**

Instead, email us at **X** with a description of the issue, steps to reproduce it, and any relevant details. We will respond as quickly as possible and work with you on a fix before any public disclosure.

## Scope

SwitchAI is a self-hosted application. The primary security concern is the protection of API keys stored in `keys.local.json` on the host machine. We take any vulnerability that could expose those keys or allow unauthorized access to a room seriously.

## Supported versions

We only maintain the latest version on the `main` branch.
